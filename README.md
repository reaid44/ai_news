README.md-# News Backend + Frontend

## Quick start
1. কপি ফাইলগুলো প্রজেক্ট ফোল্ডারে রাখো।
2. `.env.example` কপি করে `.env` বানাও এবং `NEWS_API_URL` সেট করো যদি তোমার এক্সটার্নাল API থাকে।
3. ইনস্টল করো:
   npm install
4. ডেভ মোডে চালাতে:
   npm run dev
   প্রোডাকশনে:
   npm start
5. ব্রাউজারে গিয়ে দেখো:
   http://localhost:3000
